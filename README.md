# SmartClass

A cross-platform academic management system for universities.
Built with Flutter, Node.js, and MongoDB.

## Project Structure
- `/server` — Node.js + Express REST API
- `/client` — Flutter cross-platform application

## Documentation
- Phase 1: Software Requirements Specification (SRS v1.1)
- Phase 2: System Design Document

## Tech Stack
- Frontend: Flutter (Android, iOS, Web, Desktop)
- Backend: Node.js + Express
- Database: MongoDB
- File Storage: Cloudinary
